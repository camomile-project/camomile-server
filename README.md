Camomile Annotation REST API
============================

Requirements
------------

  * node.js
  * npm
  * mongodb

Install
-------

    $ git clone git clone git@bitbucket.org:camomile_project/camomile-node.js-rest-api.git
    $ cd camomile-node.js-rest-api.git
    $ npm install

Run
---

    $ mongod
    $ node appCam.js
